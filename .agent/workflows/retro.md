# /retro

Conduct a retrospective for the current task.

Output Format:

- Keep (What went well)
- Problem (What to improve)
- Try (Action items for next time)
- Doc Health: Did this task create or reference more than 1 spec file for the same feature? If yes, flag as "Consolidation Recommended" and propose which files to merge.
